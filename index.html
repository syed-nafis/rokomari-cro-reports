Key learnings
Navigation visibility is the single highest-leverage change this month
Simply surfacing the extra-discount page in the main nav drove ×3.7 views and ৳80,269/month in incremental revenue — with no product changes. There are almost certainly other high-value, low-visibility pages that would respond the same way. Recommend a nav audit in May.
Fold-visibility for critical CTAs drives immediate revenue
Fixing the "Confirm Order" button placement above the checkout summary (Exp 11) drove a +6.73% absolute purchase rate lift. Essential actions must never be buried below the fold, especially for new users without saved shipping data.
Bangla-native CTAs outperform English on mobile
The Bangla "কার্টে যান" label drove +14.4% view-cart rate vs. +12.8% for English — a meaningful gap for a primarily Bengali-speaking audience. This pattern should be tested systematically across search, wishlist, and checkout CTAs in May.
Copy beats design — intent-driven titles outperform visual redesigns
The featured category title test (Variant C: +4.9% CTR) outperformed all design-only tests this month. Messaging that communicates clear value intent drives stronger engagement than visual styling alone. Prioritise copy testing in H2.
Micro-optimisations yield small, incremental gains
Scaling the "%OFF" discount font (Exp 12) drove a minor +0.53% purchase rate lift, while structural changes like CTA placement drove 10x that impact. We should balance fine-tuning UI details with high-leverage structural tests.
Tests under 7 days are not actionable — enforce minimum runtime
Multiple experiments ran for only 1–6 days — insufficient to achieve statistical significance or capture weekly traffic patterns. Going forward, all tests should have a minimum 7-day runtime, with a pre-set sample size calculation from GrowthBook before launch.
